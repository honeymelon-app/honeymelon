# Development Roadmap

This page provides an overview of Honeymelon's development roadmap, including the current version status and future planned features.

## Current Status

**Current Version**: 0.0.1 (Initial Public Release)
**Released**: November 2025
**Status**: Stable Foundation Release

Version 0.0.1 includes the complete core conversion pipeline, all fundamental features, and is ready for production use. See the [full roadmap](#complete-roadmap) for details on completed features.

## Next Milestones

### Version 0.1.0 – Polish & Testing Infrastructure

**Target**: Q1 2025

- Comprehensive unit test suite (Vitest)
- End-to-end testing (Playwright)
- UI/UX refinements
- Performance optimizations
- Documentation improvements

### Version 0.2.0 – Advanced Features

**Target**: Q2 2025

- Subtitle burn-in support
- Extended format support (improvements to MKV, MOV)
- Audio-only conversion
- Advanced batch workflows
- File discovery enhancements

### Version 0.3.0 – Extensibility & Integration

**Target**: Q3 2025

- Custom FFmpeg arguments editor
- Filter graph UI builder
- Basic video editing (trimming, effects)
- Automation & scripting support
- Performance tier system

### Version 1.0.0 – Stability & Polish

**Target**: Q4 2025 / Q1 2026

- 100% test coverage for critical paths
- Performance benchmarks
- Platform feature integration (Spotlight, Handoff)
- Automatic updates system
- In-app help and documentation
